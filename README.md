# Credit Scoring Model - CodeAlpha Task 1

## Project Overview
This project is a Machine Learning based Credit Scoring Model developed to predict whether a loan applicant is "Safe" or "Risky".

## Features
- **Dataset:** 32,581 records with 12 features (Income, Age, Loan Amount, etc.)
- **Preprocessing:** Median imputation for missing values.
- **Algorithms Used:** Decision Tree, Random Forest.
- **Best Model:** Random Forest Classifier.

## Performance Metrics
- **Accuracy:** 93%
- **ROC-AUC Score:** 0.93
- **F1-Score:** 0.82

## Libraries Used
- Pandas, NumPy
- Scikit-learn
- Matplotlib & Seaborn
