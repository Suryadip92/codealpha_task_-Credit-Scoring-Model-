# Credit Scoring Model - CodeAlpha Task 1

## Project Overview
This repository contains the development of a robust **Credit Scoring Model** designed to predict the creditworthiness of loan applicants. By leveraging machine learning, this system assists financial institutions in identifying potential defaulters and making data-driven lending decisions.

This project was completed as part of the **CodeAlpha** Machine Learning Internship (Task 1).

## Problem Statement
The goal is to classify loan applicants into two categories:
- **Safe (0):** Likely to repay the loan.
- **Risky (1):** High probability of default.

## Workflow
1. **Exploratory Data Analysis (EDA):** Visualizing distributions and analyzing feature correlations.
2. **Data Preprocessing:** Handled missing values (Median Imputation) and addressed class imbalance.
3. **Model Training:** Evaluated Decision Tree, and Random Forest.
4. **Performance Evaluation:** Used metrics like Accuracy, F1-Score, and ROC-AUC.

## Results
The **Random Forest Classifier** was the best performing model:
- **Accuracy:** 93%
- **AUC Score:** 0.93
- **F1-Score:** 0.82

## Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
